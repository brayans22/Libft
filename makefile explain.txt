    __________________________________________________________________________________________________________________________

                        CC = El compilador de C (GCC).
    __________________________________________________________________________________________________________________________

                        CFLAGS = flags de compilación y la instruccion -I ./ agrega el directorio actual 
                                 al camino de inclusión para buscar archivos (.h).
    __________________________________________________________________________________________________________________________

                        .c.o = Establece cómo compilar archivos (.c) a archivos objeto (.o). 
                               pasa cada archivo fuente (.c) a un archivo objeto (.o).
    __________________________________________________________________________________________________________________________

                        ${LIB}: ${OBJS} = Construye la biblioteca (.h) libft.a a partir de todos los archivos 
                                          (.o) de la lista de todos los archivos de (LIBC).
    __________________________________________________________________________________________________________________________
    
                        bonus: ${OBJSALL} = Construye la biblioteca (.h) libft.a a partir de todos los archivos 
                                            (.o) de la lista de todos los archivos de (LIBC, ADDITIONAL, BONUS).
    __________________________________________________________________________________________________________________________
    
                        all: ${LIB} = Esta regla especifica que cuando se ejecute make all, se construirá la 
                                      biblioteca normal.
    __________________________________________________________________________________________________________________________

                        clean:	rm -f ${OBJSALL} 
                        Esta regla elimina todos los archivos objeto y limpia el directorio.
    __________________________________________________________________________________________________________________________

                        Esta regla elimina la biblioteca (libft.a) y realiza la limpieza adicional 
                                que realiza la regla clean.
    __________________________________________________________________________________________________________________________

                        re: fclean all: Esta regla primero elimina todo (fclean) y luego vuelve a construir todo (all), lo que equivale a reconstruir completamente el proyecto.
    __________________________________________________________________________________________________________________________

                        .PHONY: ...: Esto declara los objetivos como "falsos", lo que evita que Make intente buscar archivos con estos nombres como objetivos.
